# 物件宣告
## 基本

```js
const card = { // 大括號物件Object
  name: "Jin" ,
  address: "Taiwan" ,
  age: 90
}
```

冒號左邊為 key，右邊為 value
亦稱為屬性及屬性值。
讀取時直接在物件名稱後面用小數點`.`接屬性名稱，或是用中括號引號`[""]`內寫key，如
```js
card.name = "Jin";
card["address"] = "Taiwan";
```

物件內也可寫入函式，為物件的方法(method)

```js
const card = { // 大括號物件Object
  name: "Jin" ,
  address: "Taiwan" ,
  age: 90,
  print: function(){
    console.log(this.name);
  },
  // or
  print(){
    console.log(this.name); 
  }
}
```
