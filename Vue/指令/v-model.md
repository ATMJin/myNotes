# v-model

雙向綁定變數。
改變原始變數的值會改變網頁中的值，改變網頁中的值也會改變原始變數的值。
只能用在`<input>`,`<select>`, `<textarea>` 或components
但`<input type="file">`不能使用

## radio / checkbox
```js
<input type="radio" name="sex" id="">
<input type="radio" name="sex" id="">
```

```js
<input type="checkbox" name="" id="">
```
