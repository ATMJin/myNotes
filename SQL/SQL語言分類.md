## 資料查詢語言 (DQL, Data Query Language)
- [[SELECT]]  查詢表格內的資料

## 資料定義語言 (DDL, Data Definition Language)
- CREATE  建立物件
- ALTER  變更物件結構
- DROP  刪除物件
- RENAME  變更物件名稱
- TRUNCATE  清除表格內的資料

## 資料操作語言 (DML, Data Manipulation Language)
- INSERT  新增表格內的資料
- UPDATE  更新表格內的資料
- DELETE  刪除表格內的資料

## 資料控制語言 (DCL, Data Control Language)
- GRANT  授予權限
- REVOKE  移除權限
